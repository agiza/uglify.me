Uglify.me
A simple web service for minifying JavaScript on the fly.
To get started, post an un-minified JavaScript file to: 
http://www.uglify.me/uglify
Note: the uglify.me service was moved from a naked domain to www.uglify.me on Dec 5, 2014. If your application is still using the naked domain you will need to update immediately to avoid issues. If you are using the Drupal module you will need to update to the latest version to continue to use the service.
Want to run your own uglify.me service? Fork on GitHub.
Uglify.me was built with love by Elliott Foster during "4K Labs" time at Four Kitchens.
